Introduction¶

This documentation provides instructions to install and customize easily a Shopinvader storefront. Frontend developers can use this guide to create custom store template for a specific usage.

Backoffice customization is out of the scope of this guide, please use Backend developper guide.

Getting Started¶

This guide was written to help you download and configure a Shopinvader template.

Setting up¶

For quick start you can use the shopinvader demo template to build your store. This template is an opensource project.
